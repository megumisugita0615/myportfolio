MegumiSugita

東京在住のWebエンジニア。 公立大学の文学部を卒業後、未経験のWebエンジニアとして新卒入社。 入社後、実際に開発しながらWeb技術を広く学習。

いつも前向き。楽しみながら取り組んでいます。

プライベートでもWebアプリ開発やインフラ領域の学習を実施。グラフィック作成・Webデザインなども勉強中。

自分の手でコツコツつくることが大好きです。

skill

フロントエンド

HTML

命名規則を用いたコーディングが可能。スタイルシートやスクリプトなどの読み込みもスムーズに実施。アプリ開発のたびに触れていて、最も慣れている言語。

CSS

BEM記法、SCSSコンパイルでの実装を経験。BootStrapのグリッドシステムを用いてレスポンシブを実装可能。

JavaScript

ハンバーガーメニューやライブラリ（particles.js）などの実装を経験。 JavaScript Standard Styleを用いたフォーマット。 アルゴリズムの組み立ても可能。

TypeScript

TypeScriptの型づけを学習し、JSアルゴリズムの書き換えを経験。TypeScriptを実行するための環境構築も可能。

React

npm・yarnでの環境構築、プロジェクトの作成方法、各ディレクトリの役割を学ぶ。技術書の理解と実際の開発を実施。

バックエンド

PHP

アルゴリズムの組み立て、オブジェクト指向プログラミングが可能。phpcs, phpcbfを用いてコードをフォーマットを実施。開発環境の構築も経験。

Laravel

Laravelフレームワークを用いたWebアプリ開発を経験。CRUD機能、画像アップロード機能、ログイン機能、API機能を実装。

Node.js

Node.jsを用いてJavaScriptをサーバーサイドで実行可能。npm/yarnでのパッケージインストール・環境構築も経験。

Express

現在、環境構築や記述法などを学習中。環境構築、CRUD機能の実装を予定。

データベース

MySQL

SQL文でDBを操作可能。PHPやNode.jsと接続したCRUD機能実装を経験。

開発

Docker

Dockerの仕組みを学習。LaravelやNode.jsなど、バックエンド環境をDocker Composeで構築。

GitHub

gitとGitHubの仕組みを学習、実践。PullRequest機能を用いた相互レビューも可能。

Visual Studio Code

使用エディタ。拡張機能やフォーマットなどのsettings.jsonを用いた設定も経験。

デザインツール

Adobe XD

現在学習中。主にワイヤーフレームの作成、Webページのデザインに使用。

Illustrator

現在学習中。本サイトのキービジュアル（アニメーション）作成に使用。

Photoshop

現在学習中。主に画像加工に使用。ゆくゆくはグラフィック作成も想定。
